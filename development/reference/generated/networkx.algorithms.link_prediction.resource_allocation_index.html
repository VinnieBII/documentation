resource_allocation_index¶

resource_allocation_index(G, u, v)[source]¶

Compute the resource allocation index of u and v.

Resource allocation index of nodes u and v is defined as the sum of reciprocals of the degree of all the common neighbors w of u and v.

Parameters :

G : graph

A NetworkX undirected graph.

u, v : nodes

Nodes in the graph.
Returns :

value : float

The resource allocation index of u and v.

References

[R252]T. Zhou, L. Lu, Y.-C. Zhang. Predicting missing links via local information. Eur. Phys. J. B 71 (2009) 623. http://arxiv.org/pdf/0901.0553.pdf

Examples

>>> G = nx.complete_graph(5)
>>> nx.resource_allocation_index(G, 0, 1)
0.75
